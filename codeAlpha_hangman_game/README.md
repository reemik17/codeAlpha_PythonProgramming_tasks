# Hangman Game
Description
This project is a simple text-based Hangman game written in Python. A random fruit is selected from a predefined list, and the player attempts to guess the word one letter at a time. The game ends when the player either guesses the entire word or reaches the maximum number of incorrect guesses.

**Features**
* Random word selection
* Tracks guessed letters
* Displays the current progress after every guess
* Prevents duplicate and invalid inputs
* Allows up to six incorrect guesses
* Displays the correct word when the game ends

**Concepts Practiced**
* Random module
* Lists
* Strings
* Loops
* Conditional statements
* Input validation
