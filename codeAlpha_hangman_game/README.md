# Hangman Game

## Description
A text-based word guessing game where the player uncovers a randomly selected fruit by guessing one letter at a time within a limited number of attempts.

## Features
- Randomly selects a fruit from a predefined list
- Tracks guessed letters
- Displays the current progress after each guess
- Prevents duplicate and invalid inputs
- Allows up to six incorrect guesses
- Reveals the correct word when the game ends

## Concepts Practiced
- Random module
- Lists
- Strings
- Loops
- Conditional statements
- Input validation
