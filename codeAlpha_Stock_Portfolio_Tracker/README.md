# Stock Portfolio Tracker
This project calculates the value of a stock portfolio using predefined stock prices. Users enter stock symbols and the number of shares they own. The program calculates the total investment value, displays a portfolio summary, and saves the results to a text file.

## Features
- Uses predefined stock prices
- Accepts multiple stock entries
- Validates share quantities
- Displays a portfolio summary
- Calculates the total investment value
- Saves the portfolio summary and total investment to a text file

## Concepts Practiced
- Dictionaries
- Loops
- User input/output
- Arithmetic calculations
- Exception handling (try-except)
- File handling
