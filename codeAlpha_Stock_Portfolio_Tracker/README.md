# Stock Portfolio Tracker
This project calculates the value of a stock portfolio using predefined stock prices. Users enter stock symbols along with the number 
of shares they own, and the program computes the total investment value before saving the result to a text file.

**Features**
* Uses predefined stock prices
* Accepts multiple stock entries
* Validates share quantities
* Calculates the total portfolio value
* Stores the final result in a text file

**Concepts Practiced**
* Dictionaries
* Loops
*  User input/output
* Arithmetic calculations
* File handling
