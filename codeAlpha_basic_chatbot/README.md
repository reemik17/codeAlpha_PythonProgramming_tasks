# Basic Chatbot
Description
This project is a simple chatbot that responds to predefined user messages. It demonstrates how basic conversational programs can be created using conditional statements and loops. The chatbot continues interacting until the user chooses to exit.

Features
1) Responds to common greetings and questions
2) Provides predefined replies
3) Handles unknown inputs gracefully
4) Runs continuously until the user enters bye

Concepts Practiced
1) Functions
2) Loops
3) Conditional statements
4) String manipulation
5) User input/output
