# Basic Chatbot

## Description
This project is a simple rule-based chatbot built in Python. It responds to predefined user messages using conditional statements and continues interacting with the user until the exit command is entered.

## Features
- Responds to common greetings and questions
- Provides predefined responses
- Handles unknown inputs gracefully
- Runs continuously until the user enters **bye**

## Concepts Practiced
- Functions
- Loops
- Conditional statements
- String manipulation
- User input/output
